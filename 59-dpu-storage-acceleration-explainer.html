DPU Storage Acceleration Overview

DPU storage acceleration enables high-performance storage access through NVMe over Fabrics (NVMe-oF) and composable storage architectures. DPUs can accelerate storage operations by offloading NVMe-oF processing, enabling remote storage access with local-like performance. Composable storage allows dynamic allocation of storage resources to compute nodes, providing flexible and efficient storage utilization. This enables disaggregated storage architectures where storage is separated from compute, enabling better resource utilization and scalability.

NVMe over Fabrics (NVMe-oF)

NVMe-oF enables remote access to NVMe storage devices over network fabrics (InfiniBand, Ethernet, Fibre Channel). DPUs can accelerate NVMe-oF operations by offloading protocol processing, enabling high-performance remote storage access.

Composable Storage

Composable storage enables dynamic allocation of storage resources to compute nodes. DPUs enable composable storage by providing the infrastructure to dynamically attach and detach storage resources, enabling flexible and efficient storage utilization.
