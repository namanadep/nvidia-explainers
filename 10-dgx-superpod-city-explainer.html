DGX SuperPOD Architecture Overview

DGX SuperPOD is NVIDIA's reference architecture for building large-scale AI training infrastructure. It provides a complete, integrated solution that combines compute nodes, high-performance networking, storage systems, and management software. The architecture is designed for scalability, enabling organizations to start with a BasePOD foundation and expand to SuperPOD scale as requirements grow. All components are pre-validated and optimized to work together seamlessly.

DGX Nodes: Compute Units

DGX nodes are individual compute units in the SuperPOD cluster. Each node contains multiple GPUs and provides compute resources for AI workloads. Nodes work together as part of the cluster infrastructure, sharing network, storage, and management systems.

Node Roles and Workloads

Each DGX node can host different types of workloads: some nodes run training jobs, others handle inference, and others process data. All nodes connect to shared cluster infrastructure including high-speed network (InfiniBand), parallel storage (Lustre), and management systems (BCM). This shared infrastructure enables efficient resource utilization and workload distribution.

Interconnect: NVLink & NVSwitch

NVLink and NVSwitch provide high-speed interconnect for GPU communication within and between nodes. NVLink connects GPUs within a node, while NVSwitch enables all-to-all communication between GPUs across nodes, enabling efficient distributed training.

Interconnect Architecture: NVLink provides high-speed interconnect between GPUs within a node. NVSwitch enables high-speed interconnect between nodes. Together, they create an all-to-all communication network where any GPU can communicate with any other GPU at maximum bandwidth and minimum latency.

SuperPOD is designed to scale seamlessly - network, storage, and management infrastructure accommodate growth while maintaining performance and operational efficiency.
