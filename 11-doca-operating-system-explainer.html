DOCA Overview

DOCA (Data Center Infrastructure-on-a-Chip Architecture) is NVIDIA's comprehensive software framework for developing, deploying, and managing applications on BlueField DPUs. The DPU hardware provides powerful compute resources including ARM cores, high-speed network interfaces, and dedicated memory, but requires software to enable its capabilities. DOCA provides the operating system, runtime environment, and development tools needed to transform DPU hardware into a programmable infrastructure platform.

DOCA SDK

The DOCA SDK provides comprehensive development tools for creating custom DPU applications. It includes libraries with pre-built functions for common DPU operations, APIs for hardware access, development and debugging tools, documentation, and sample applications. The SDK enables developers to build custom infrastructure applications that leverage DPU capabilities.
