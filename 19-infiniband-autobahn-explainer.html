InfiniBand Overview

InfiniBand is a high-performance networking technology designed specifically for data center environments requiring ultra-low latency, high bandwidth, and lossless data transmission. Unlike traditional Ethernet, which was designed for general-purpose networking, InfiniBand is optimized for the communication patterns found in AI training and HPC workloads, where massive amounts of data must be transferred between compute nodes with minimal delay.

InfiniBand vs Ethernet

Traditional Ethernet provides functional networking capabilities but includes overhead from protocol layers, congestion management, and variable latency characteristics. While suitable for many applications, Ethernet's design limitations make it less optimal for AI workloads that require consistent, ultra-low latency and guaranteed bandwidth for collective communication operations.

RDMA: Remote Direct Memory Access

RDMA (Remote Direct Memory Access) enables direct memory-to-memory data transfer without CPU involvement, significantly reducing latency and CPU overhead. This makes data transfer much faster and more efficient than traditional network protocols.

RDMA Benefits: Traditional networking requires CPU involvement for data transfer, adding latency and consuming CPU cycles. RDMA bypasses the CPU entirely, allowing direct memory-to-memory transfer. This zero-copy operation reduces latency to microseconds and eliminates CPU overhead, making it ideal for high-performance distributed training workloads.

Subnet Manager

The Subnet Manager discovers network topology, calculates optimal routes, and assigns identifiers (LIDs for local routing, GIDs for global routing). It maintains network state and handles topology changes, ensuring efficient packet routing throughout the InfiniBand fabric.

Flow Control

Credit-based flow control prevents congestion by requiring receivers to grant credits before senders transmit data. This ensures data flows smoothly without packet loss, maintaining InfiniBand's lossless characteristics. Flow control prevents buffer overflow and maintains predictable performance.

UFM: Unified Fabric Manager

UFM (Unified Fabric Manager) provides comprehensive network monitoring, management, and optimization for InfiniBand fabrics. It ensures the network operates efficiently and identifies issues before they impact workloads.
