Clustered NAS Overview

Clustered NAS (Network Attached Storage) provides shared file storage across multiple storage nodes, creating a scalable, high-performance storage system for AI clusters. Unlike traditional single-node NAS, clustered NAS distributes data and metadata across multiple nodes, enabling better performance, scalability, and reliability. Clustered NAS architectures are designed to handle the high-throughput, concurrent access patterns typical of AI training workloads, making them well-suited for AI infrastructure.

Clustered NAS: Clustered NAS is like a distributed library system where books (data) are stored across multiple library branches (storage nodes), but all branches share a unified catalog (namespace). This allows multiple readers (compute nodes) to access books simultaneously from different branches, providing better performance and scalability than a single large library.

Performance Characteristics

Clustered NAS performance characteristics make it suitable for AI workloads, with performance that scales with the number of storage nodes.

Throughput

  • Aggregate Throughput: Sum of all node throughputs
  • Scalable: Throughput scales with nodes
  • High Performance: Can achieve 100+ GB/s aggregate
  • Concurrent Access: Supports concurrent access

Latency

  • Low Latency: Low latency for metadata operations
  • Consistent: Consistent latency characteristics
  • Optimized: Optimized for AI workloads
  • Network Dependent: Latency depends on network

Scalability

  • Capacity Scaling: Capacity scales with nodes
  • Performance Scaling: Performance scales with nodes
  • Linear Scaling: Near-linear scaling characteristics
  • Large Scale: Supports large-scale deployments
Clustered NAS Selection: Choose clustered NAS for AI workloads requiring shared storage with high performance and scalability. Clustered NAS provides better performance and scalability than single-node NAS, making it suitable for medium to large AI clusters. Consider clustered NAS when you need shared file storage with performance that scales with cluster size.
